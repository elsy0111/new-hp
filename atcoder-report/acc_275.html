A - Find Takahashi AC

- 問題 -

AtCoder 村には $N$ 本の橋があり、$i$ 本目（ $1\leq i\leq N$ , $i \in \mathbb{R}$ ） の橋の高さは $H_i$です。
ここで、AtCoder 村にある N 本の橋のうち、どの相異なる $2$ 本の橋も高さが異なります。
AtCoder 村で最も高い橋は何本目の橋か出力してください。

制約

$1\leq N\leq 100$
$1\leq H_i \leq 10^9$
$H_i$ は全て異なる
$N,H \in \mathbb{Z}$

入力

入力は以下の形式で標準入力から与えられる。

$N$
$H_1\quad H_2\quad ...\quad H_N$

出力

AtCoder 村で最も高い橋は何本目の橋かを、整数で出力せよ。

入力例

3
50 80 70

出力例

2

- 解説 -

問題文の要約

$max(h)$ のインデックス。
$\large \Rightarrow \overset{\Large argmax(h_i)}{\small i \quad}$
便利なことに、この $argmax$ をそのまま実装できるライブラリが numpy にあります。
それを使って実装してみましょう。

Code

import numpy as np

n = int(input())
h = list(map(int,input().split()))

print(np.argmax(h) + 1)

ほかの言語がかわいそうになるくらい簡潔に実装できましたね...

B - ABC-DEF AC

- 問題 -

非負整数 $A,B,C,D,E,F$ があり、$A×B×C\geq D×E×F$ をみたしています。
$(A×B×C)−(D×E×F)$ の値を $998244353$ で割った余りを求めてください。

制約

$1\leq A,B,C,D,E,F\leq 10^{18}$
$A×B×C \geq D×E×F$
$A,B,C,D,E,F \in \mathbb{Z}$

入力

入力は以下の形式で標準入力から与えられる。

$A\quad B\quad C\quad D\quad E\quad F\quad$

出力

$(A×B×C)−(D×E×F)$ の値を $998244353$ で割った余りを整数で出力せよ。

入力例

2 3 5 1 2 4

出力例

22

- 解説 -

問題文の要約

圧倒的 B<<A。
要約も何もないです。

Code

a,b,c,d,e,f = list(map(int,input().split()))

print(((a*b*c)-(d*e*f))%998244353)
C - Counting Squares WA

- 問題 -

二次元平面があります。$1$ 以上 $9$ 以下の整数 $r,c$ について、$S_r$ の $c$ 番目の文字が # であるとき座標 $(r,c)$ にポーンが置いてあり、$S$ の $c$ 番目の文字が . であるとき座標 $(r,c)$ に何も置かれていません。
この平面上の正方形であって、$4$ 頂点全てにポーンが置いてあるものの個数を求めてください。

制約

$S_1,S_2,...,S_9$ はそれぞれ # と . からなる長さ $9$ の文字列

入力

入力は以下の形式で標準入力から与えられる。

$S_1$
$S_2$
$\vdots$
$S_9$

出力

答えを出力せよ。

入力例

##.......
##.......
.........
.......#.
.....#...
........#
......#..
.........
.........

出力例

2

- 解説 -

問題文の要約

# を頂点とする正方形は何個ありますか？
ここで判定するポイントとしては $2$ 個の頂点が決まった時点で残りの頂点は決まります。その個所に # があるのかどうかを調べます。
全ての座標において説明するのは難しいので、以下は # が $4$ 個であり、 即ち、正方形が $1$ この場合について説明します。
$1$ 個目の # の座標を $(x_1,y_1)$
$n$ 個目の # の座標を $(x_n,y_n)\quad (1 \leq n \leq 4)$
$1$ 個目の # の座標と $2$ 個目の # の座標の差を $(\Delta x,\Delta y)$ と表すこととします。
$\:\Leftrightarrow (\Delta x,\Delta y) = (x_2,y_2) - (x_1,y_1)$

したがって、求めるのは以下です。

$(x_1,y_1)$ $\stackrel{\text{def}}{=} (x,y)$
$s.t. (x,y) =$ #

$\alpha : \: (x_2,y_2)$ $=(x_1 + \Delta x,y_1 + \Delta y)$ $=$ #
$\beta : \: (x_3,y_3)$ $=(x_1 - \Delta y,y_1 + \Delta x)$ $=$ #
$\gamma : \: (x_4,y_4)$ $=(x_1 - \Delta y + \Delta x,y_1 + \Delta x + \Delta y)$ $= $ #

$\alpha \land \beta \land \gamma$ を満たすとき, $(x_n,y_n) = $ # を頂点とする正方形であるといえる。

Code [WA]

arr = []  #入力を格納する配列
for i in range(9):
    arr.append(input())
 
ans = 0   #条件を満たした正方形の個数

for i in range(9):
    for j in range(9):
        if arr[i][j] == "#":
            break_bool = False
            x1 = i
            y1 = j
            for y in range(y1,9):
                for x in range(9):  #↓(x_1,y_1)よりも右下を探索 => (x_1,x_2)までは探索しない
                    if y == y1 and x <= x1:
                        continue
                    if arr[x][y] == "#":  #(x_2,y_2) = #
                        dx = x - x1
                        dy = y - x2
                        break_bool = True
                        break
                if break_bool:
                    break
            if not break_bool:
                break
            
            try:
                if arr[x1 + dx][y1 + dy] == "#":
                    if arr[x1 - dy][y1 + dx] == "#":
                        if arr[x1 - dy + dx][y1 + dy + dx] == "#":
                            ans += 1
            except IndexError:
                break
 
print(ans)

なぜ間違えたのですか?

$(x_1,y_1)$ を頂点とする正方形は1つだけではないのです...

D - Yet Another Recursive Function TLE

- 問題 -

非負整数 $x$ に対し定義される関数 $f(x)$ は以下の条件を満たします。

$f(0) = 1$
任意の正整数 $k$ に対し $f(k)=f({\large \lfloor \frac{k}{2} \rfloor} ) + f({\large \lfloor \frac{k}{3} \rfloor })$

ここで、$\lfloor A \rfloor$ は $A$ の小数点以下を切り捨てた値を指します。
このとき、 $f(N)$ を求めてください。

制約

$0\leq N\leq 10^{18}$
$N \in \mathbb{Z}$

入力

入力は以下の形式で標準入力から与えられる。

$N$

出力

答えを出力せよ。

入力例

2

出力例

3

- 解説 -

問題文の要約

全ての項が$f(0)$になるまで関数を呼び出して計算してください。

そんな膨大な計算量どうすればいいですか

メモ化を使うといいです。
関数の引数に関数があり、その引数にも関数が、、このような構造、現象のことを再帰といいます。
再帰をする問題ではよくメモ化と呼ばれる手法が出てきます。
メモ化というのは一度計算した値をメモしておくことで計算量を劇的に削減する手法のことです。
例としてはフィボナッチ数列の計算にも用いられています。
