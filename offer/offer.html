AtCoderは、世界最高峰の競技プログラミングサイトです。
リアルタイムのオンラインコンテストで競い合うことや、
3,000以上の過去問にいつでもチャレンジすることができます。
引用:AtCoder.jp

実際に何をするのかというと、
週に一回コンテストに参加し、一斉に問題を解き、その速さや正誤に応じてポイントが付き、その結果は順位やレートに反映されます。
レートとか聞いたらゲームっぽくていいでしょ?笑

難易度は簡単なものからA問題,B問題,...といって、100,200,...というようにポイントが降られています。
同点だった場合に提出の速さ、誤った提出をした場合のペナルティなどをもとに順位が算出されます。

以下は「AtCoder記録簿」として私が解説したものです。
From : AtCoder Beginner Contest 274 - B

B - Line Sensor AC

- 問題 -

縦 $H$ マス、横 $W$ マスのグリッドがあります。上から $i$ 行目、左から $j$ 列目のマスを $(i,j)$ と表します。 各マスの状態は文字 $C_{i,j}$ で表されます。 $C_{i,j}$ が . ならば $(i,j)$ には何も置かれておらず、 # ならば箱が $1$ 個置かれています。
$1\leq j\leq W$ を満たす整数 $j$ に対して、整数 $X_j$ を次のように定義します。

$j$ 列目に置かれている箱の個数。言い換えると、$C_{i,j}$ が # であるような整数 $i$ の個数。

$X_1,X_2,...,X_W$ をすべて求めてください。

制約

$1\leq H\leq 1000$
$1\leq W\leq 1000$
$H,W \in \mathbb{Z}$
$C_{i,j} \in \{\hspace{7px} .,\verb|#| \hspace{7px} \}$

入力

入力は以下の形式で標準入力から与えられる。

$H\quad W$
$C_{1,1}C_{1,2} ... C_{1,W}$
$C_{2,1}C_{2,2} ... C_{2,W}$
$\hspace{7px} \vdots$
$C_{H,1}C_{H,2} ... C_{H,W}$

出力

$X_1,X_2,...,X_W$ を以下の形式に従って出力せよ。

$X_1\quad X_2\quad ...\quad X_W$

入力例

3 4
#..#
.#.#
.#.#

出力例

1 2 0 3

- 解説 -

問題文の要約

縦横のテーブルの縦(列)に、#は何個あるか。

$\Rightarrow$ 二次元配列を作り、各行の同じ個数目の要素に#は何個あるか。

Code

h,w = map(int,input().split())

//arr : 縦h行*横w列の二次元配列
arr = []
for i in range(h):
    arr.append(input())

ans = []                //解答の格納
for j in range(w):      //全(w)列回す
    cnt = 0
    for i in range(h):  //全(h)行回す
        if arr[i][j] == '#':
            cnt += 1
    ans.append(cnt)

//出力
for i in ans[:len(ans) - 1]:
    print(i,end = " ")
print(ans[-1])
